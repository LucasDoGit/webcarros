## Introdução

Projeto de um site de venda e compra de veículos, baseado no site da [Web Motors](https://www.webmotors.com.br). Desenvolvido com objetivo de práticar principalmente os conhecimentos obtidos em react, typescript e firebase.

## Como Executar o Projeto

Para executar o projeto siga as instruções:

1. Clone o repositório:

```bash
git clone https://github.com/LucasDoGit/webcarros
cd webcarros
```

2. Instale as dependências:

```bash
npm install
```

3. (Opcional) configurar o arquivo `src/services/firebaseConnection.ts` com as suas credenciais do seu projeto no firebase:

```typescript
const firebaseConfig = {
  apiKey: "##",
  authDomain: "##",
  projectId: "##",
  storageBucket: "##",
  messagingSenderId: "##",
  appId: "##",
};
```

4. use o comando abaixo para iniciar o projeto e aguarde.

```bash
npm run dev
```

5. Feito! O projeto deve ser iniciado e pode ser executado em um navegador.
